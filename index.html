Vantagens

Mobilidade

Com o intuito de manter as manutenção e gastos do veículo em dias e em mãos o CarHere atende o requisito de mobilidade facilitando a vida do condutor. O CarHere no momento só está disponível para Android

Gerenciamento

Com possibilidade de cadastramentos de serviços realizados e autonomias, ficando mais fácil gerenciar o que já foi realizado e o que tem a ser realizado, o aplicativo traz uma média de tempo a se realizar novos serviços com base nos dados cadastrados

Banco de Dados

CarHere utiliza um banco de dados em nuvem onde fica armazenado os dados dos usuarios proporcionando assim mais segurança e também conta com um banco de dados offline onde o usuário pode estar utilizando o APP mesmo não estando na rede e após tiver conexão os seus dados serão transferidos para a nuvem

Sobre

CarHere

  • 2020

    Inicio

    Nasce uma ideia de projeto de faculade onde o intuito do programa é realizar o gerenciamento de veiculos

  • 2021

    Desenvolvimento

    Após ter a toda a ideia construida e prototipos feitos seguimos para iniciar a parte da programação do APP o qual o nome seria MeuPossante

  • Inicio de 2022

    Refinamento

    Foi alterado o nome do APP para CarHere, finalizado as principais funcionalidades do APP e realizado os ajustes de validação

  • Final de 2022

    Finalização

    Feita a integração com o Firebase e demais melhorias no APP como utiliziação de API para deixar o app mais simples de ser utilizado
